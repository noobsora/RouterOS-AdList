# 贡献指南 (CONTRIBUTING)

感谢你关注使用 RouterOS-AdList ！  

## 添加新的域名源

欢迎推荐更多优质的去广告域名源！  
如果你有想要添加的源，请：

- 先检查该源是否公开、稳定且格式合适；  
- 在 GitHub 仓库的 [Issues](https://github.com/noobsora/RouterOS-AdList/issues) 中打开一个新 Issue，说明源的名称和地址；  
- 简单介绍该源的特点及为何适合加入本项目。

我会定期查看 Issue，并评估是否纳入项目。

## 反馈问题和建议

如果在使用过程中发现任何问题或者有改进建议，欢迎在 Issue 里提交，描述尽量详细。
